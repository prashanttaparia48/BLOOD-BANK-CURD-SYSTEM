# BLOOD-BANK-CURD-SYSTEM
# DESCRIPTION
A SIMPLE PYTHON BASED PROJECT TO ADD, STORE, UPDATE AND MANAGE THE DATA OF BLOOD BANK

## FEATURES 
  - ASKS THE USER THE REQUIRED BLOOD TYPE
  - VIEW THE CURRENT INVENTORY
  - ADD NEW BLOOD STOCK TO INVENTORY
  - DISPENSE BLOOD UNITS
  - SHOWS COMPATIBLE BLOOD TYPES ACCORDING TO THE REQUIRED BLOOD TYPE
  - ASKS THE USER THE REQUIRED BLOOD TYPE
  - UPDATES INVENTORY AFTER DISPENSING BLOOD UNITS

## TECHNOLOGY USED 
 - PYTHON
 - CSV FILE HANDLING

## HOW TO RUN 
1.DOWNLOAD THE REPOSITORY
2.OPEN 'blooddndprgm.py' in any python IDE or VS CODE 
3. RUN THE PROGRAM USING:
    ```bash
    python blooddndprgm.py

## project file
   blooddndprgm.py - main code
   blood_inventory.csv - data storage file
